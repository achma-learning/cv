1. Use CSS Media Queries for Mobile Responsiveness

The most effective way to achieve this is to utilize CSS media queries to detect when a screen size is small enough (typically under 768px wide) and apply mobile-specific styling (such as removing padding and adjusting the layout). Here's an optimized approach:

    }
